# E-Commerce Sales Analysis using Microsoft Excel

## Project Overview

This project focuses on performing an end-to-end **E-Commerce Sales Data Analysis** using **Microsoft Excel**. The objective of this project is to clean, transform, analyze, and visualize sales-related data to generate meaningful business insights through descriptive statistics, pivot tables, charts, and dashboard creation.

The project demonstrates practical Excel skills used in real-world business environments for analyzing customer behavior, sales performance, product trends, payment methods, and store performance.

The complete workflow includes:

* Data Cleaning
* Data Transformation
* Descriptive Statistics
* Pivot Table Analysis
* Data Visualization using Charts
* Interactive Dashboard Creation

This project was developed as a hands-on learning experience to strengthen practical knowledge in **Microsoft Excel for Data Analysis**.

---

## Project Objectives

The key objectives of this project are:

* To clean and prepare raw e-commerce datasets for analysis
* To identify and handle missing values
* To remove duplicate records and correct inconsistencies
* To transform and structure data for meaningful reporting
* To perform descriptive statistical analysis
* To analyze sales, products, stores, and customer behavior using Pivot Tables
* To visualize insights through charts and graphs
* To build an interactive dashboard for business reporting
